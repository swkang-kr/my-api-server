  🚀 Railway 배포 방법 요약

  3가지 방법 중 선택:

  1️⃣ Railway Dashboard (가장 쉬움) ⭐

  1. https://railway.app/dashboard 접속
  2. 프로젝트 선택 → Variables 탭
  3. .env 파일 내용을 복사해서 하나씩 추가
  4. Deploy 또는 GitHub 연동 시 자동 배포

  2️⃣ Railway CLI (빠름)

  npm install -g @railway/cli
  railway login
  railway variables set DB_URL="jdbc:postgresql://..."
  railway variables set DB_USERNAME="postgres"
  railway variables set DB_PASSWORD="your-password"
  railway up

  3️⃣ 자동 스크립트 (매우 빠름)

  # Windows
  railway-setup.bat

  # Linux/Mac
  ./railway-setup.sh
