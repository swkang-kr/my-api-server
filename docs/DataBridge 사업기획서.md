## 사업 개요

### 핵심 컨셉
**"개발자 없이 5분 만에 API 연결하는 노코드 자동화 플랫폼"**

### 타겟 고객
1. **1차 타겟**: 스타트업 마케터/운영자 (개발팀 없음)
2. **2차 타겟**: 중소기업 관리자 (IT 예산 부족)
3. **3차 타겟**: 1인 사업자 (쇼핑몰, 서비스업)

### 핵심 가치 제안
- 반복 작업 자동화로 **시간 절약 80%**
- 개발자 고용 없이 **API 연동 구현**
- 한국 서비스 특화로 **즉시 사용 가능**
